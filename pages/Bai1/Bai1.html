Cài đặt XAMPP cho Windows:

Giao diện hoạt động của XAMPP

Truy cập Localhost

Tạo CSDL và bảng

Mở http://localhost/phpmyadmin → tạo database users với collation utf8_unicode_ci, rồi tạo 2 bảng theo đúng cấu trúc trong tài liệu (có thể dán trực tiếp các lệnh dưới vào tab SQL).

-- Bảng users
CREATE TABLE `users` (
  `id` int(11) unsigned NOT NULL AUTO_INCREMENT,
  `username` varchar(50) COLLATE utf8_unicode_ci NOT NULL,
  `password` varchar(50) COLLATE latin1_swedish_ci NOT NULL,
  `mail` varchar(50) COLLATE latin1_swedish_ci NOT NULL,
  `salt` varchar(255) NOT NULL,
  PRIMARY KEY (`id`)
);

-- Bảng ip
CREATE TABLE `ip` (
  `id` int(11) unsigned NOT NULL AUTO_INCREMENT,
  `ip` varchar(50) COLLATE latin1_swedish_ci NOT NULL,
  `count_login` int(11) NOT NULL,
  `created_at` timestamp NULL DEFAULT NULL,
  `updated_at` timestamp NULL DEFAULT NULL,
  PRIMARY KEY (`id`)
);

Chuẩn bị tên miền nội bộ (hosts)

Để truy cập 3 site theo yêu cầu bài tập, thêm bản ghi hosts (trên Windows là tệp C:\Windows\System32\drivers\etc\hosts).

  • Mở Notepad Run as Administrator
  • Mở tệp: C:\Windows\System32\drivers\etc\hosts
  • Thêm các dòng:
127.0.0.1   www.abc.com.vn
127.0.0.1   bulletin.abc.com.vn
127.0.0.1   hrm.abc.com.vn

Tạo 3 VirtualHost cho 3 website

Trên Windows với XAMPP, các file cấu hình Apache nằm trong C:\xampp\apache\conf\, file vhost ở C:\xampp\apache\conf\extra\httpd-vhosts.conf.

Bật include vhosts

Mở C:\xampp\apache\conf\httpd.conf → đảm bảo dòng sau không bị comment:

Include conf/extra/httpd-vhosts.conf

Tạo thư mục mã nguồn

  • C:\xampp\htdocs\abc\
  • C:\xampp\htdocs\bulletin\
  • C:\xampp\htdocs\hrm\

Mỗi thư mục tạo 1 file index.htm hoặc index.php (ví dụ <?php echo "ABC site"; ?>) để kiểm tra.

Khai báo VirtualHost

Mở C:\xampp\apache\conf\extra\httpd-vhosts.conf, thêm 3 khối:

# www.abc.com.vn
<VirtualHost *:80>
    ServerName www.abc.com.vn
    DocumentRoot "C:/xampp/htdocs/abc"
    <Directory "C:/xampp/htdocs/abc">
        AllowOverride All
        Require all granted
    </Directory>
    ErrorLog "logs/abc-error.log"
    CustomLog "logs/abc-access.log" common
</VirtualHost>

# bulletin.abc.com.vn
<VirtualHost *:80>
    ServerName bulletin.abc.com.vn
    DocumentRoot "C:/xampp/htdocs/bulletin"
    <Directory "C:/xampp/htdocs/bulletin">
        AllowOverride All
        Require all granted
    </Directory>
    ErrorLog "logs/bulletin-error.log"
    CustomLog "logs/bulletin-access.log" common
</VirtualHost>

# hrm.abc.com.vn
<VirtualHost *:80>
    ServerName hrm.abc.com.vn
    DocumentRoot "C:/xampp/htdocs/hrm"
    <Directory "C:/xampp/htdocs/hrm">
        AllowOverride All
        Require all granted
    </Directory>
    ErrorLog "logs/hrm-error.log"
    CustomLog "logs/hrm-access.log" common
</VirtualHost>
